# The Latin Experiment

A static skeleton for an Ecclesiastical Latin learning companion inspired by a modern course dashboard.

## Current scope

- Home, Course, Schedule, Materials, Certificates and Contact pages
- 81 lesson placeholders based on the first volume of Father William Most's *Latin by the Natural Method*
- An examination module after every five lessons
- Local browser progress, filters, schedule goals and indicative achievement badges
- GitHub Pages deployment workflow

This is an early visual and interaction skeleton. Course content, rights review, contact details and formal editorial review remain to be completed.
